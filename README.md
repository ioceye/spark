# 02. Spark项目
- spark-scala
	> spark开发的scala版本栗子。
	> 
	> 主要有spark-core、spark-sql、spark-hive、spark-stream等栗子。
- spark-java
	> spark开发的java版本栗子，与scala版本类似。
- spark-summary
	> 数据量：2亿 * 7 = 14亿，容量：4G * 7 = 28G。
	> 
	> 应用spark开发对其指标的汇总，其中包括spark-core、spark-dataset等操作。
	> 
	> 应用包含各种优化技术，包括spark优化和jvm优化。
	> 
	> 结果：40G内存，耗时1h。jvm gc时间小。
